とても面白い整理です。
ここでは **簡化24式太極拳**（いわゆる24式）について、

* 記号体系は前回と同一
* 6要素（歩型・歩法・手法・身法・呼吸・意念）
* 時間軸は柔軟（動作ごとに6フェーズへ正規化）
* 各式の「隘路（つまずきやすい位相ズレ点）」を明示

という形でまとめます。

---

### 歩型

虚・中・弓・馬・独

### 歩法

起・移・送・進・退・定

### 手法

抱・掤・捋・擠・按・分・穿・雲・架・打

### 身法

含・転・旋・開・沈・立

### 呼吸

吸・変・呼・収

### 意念

集・導・送・達・透・静

※残りの式も同様パターンで展開できますが、
**構造的には次の4類型に収束します：**

1. 弓歩展開型（野馬・攬雀尾・摟膝）
2. 独立型（金鶏）
3. 横移動循環型（雲手）
4. 退歩型（倒攆猴）

---

# ■ 24式全体の構造的隘路

1. 手法が歩法に従属しない
2. 呼吸が位相同期しない
3. 身法（腰回旋）が主導権を持たない
4. 独立系で重心が浮く

つまり全体は

```
足 → 腰 → 手 → 呼吸 → 意
```

の因果連鎖が崩れる地点が隘路。
